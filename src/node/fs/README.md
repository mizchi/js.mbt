# mizchi/js/node/fs

## node:fs

Sync APIs and Promise APIs

- node:fs
  - [x] readFileSync
  - [x] writeFileSync
  - [x] appendFileSync
  - [x] readdirSync
  - [x] statSync
  - [x] mkdirSync
  - [x] rmSync
- node:fs/promises
  - [x] readFile
  - [x] writeFile
  - [x] appendFile
  - [x] readdir
  - [x] stat
  - [x] mkdir
  - [x] rm
  - [x] cp

## Not supported

- Callback APIs
  - Use `mizchi/js/fs/promises` for Promise-based APIs
